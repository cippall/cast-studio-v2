# Cast Studio v2

Multi-tenant digital casting and wardrobe library.

## Project Status

This is the initial specification stage. No code yet — just specs.

## Specs

- `specs/spec.md` — System specification (what + why)
- `specs/database-schema.md` — Database schema (how we store)
- `specs/api.md` — REST API design (how we communicate)
- `specs/ui.md` — UI/UX design (how users interact)

## Tech Stack

- Backend: Node.js + Express, TypeScript ESM
- Frontend: React + TypeScript
- Database: PostgreSQL
- Image Generation: fal.ai API
- Image Storage: fal.ai URLs (primary) + local backup (future: AWS S3)
